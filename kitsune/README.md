# 概述

一个简单的萌化mod，添加了许多来自凪白老师作品的种族肖像和载入图片。
这些作品描绘了伊德娅和她的小伙伴们的故事：
巫女，飞龙，同伴，旅行，残旧废墟，远方的城市……
好吧更多的请自行脑补。
