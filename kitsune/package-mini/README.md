  * 飞龙（爬行类）：魁梧奇伟 3，四海为家 1，延年益寿 1，落落寡合 -1，缓慢增殖 -2

[h2]起源[/h2]

共同进化

* 条件：必须是某种亲外主义，不能是某种威权主义
* 效果：
  * 游戏开始时拥有12第二种族人口
  * 工作产生的社会学研究：+10%
  * 每月凝聚力：+5%

[h1]FAQ[/h1]

[b]问：请问这是关于小狐狸（白上吹雪）的mod吗？[/b]
答：不是，这是关于伊德娅和她的小伙伴们的mod，伊德娅是白上吹雪的创作者凪白老师的看板娘。

[b]问：我觉得白上吹雪和江风都非常可爱，可以一同加入她们的种族肖像和载入图片吗？[/b]
答：不行，伊德娅天下第一可爱！如有需要，请另外订阅相关mod。

[b]问：能不能告诉我其他那些角色的名字啊？[/b]
答：部分角色的名字可以参见mod中的名称列表，至于其他角色则完全不知道任何情报，因为凪白老师根本没说……
